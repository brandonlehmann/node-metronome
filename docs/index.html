Node-Metronome

A simple Metronome like repeating timer with events.

TypeScript

 
import {Metronome} from 'node-metronome';

const timer = new Metronome(1000, true);

timer.on('tick', () => {
    // do something
})

Javascript

 
const Metronome = require('node-metronome').Metronome

const timer = new Metronome(1000, true);

timer.on('tick', () => {
    // do something
})
